      # Cruise Line Berthing Interface

      # Project Summary

      • Web service client in C#, WCF
      • Interfacing constraints with a legacy application
      • Conforming to WSDL without access to the service endpoint

      # Application Outline

      The Berthing Interface (Fidelio) transfers crew berth assignments from the MAPS Maritime Rostering application to the on-board Fidelio key-pass system.

      Berthing assignments were added to MAPS as a custom module so that crew accommodation could be assigned at the time they were rostered on to a vessel.

      The Fidelio system is an on-board database of crew assignments which is attached to a key-card printer. The process is to print boarding passes for crew embarking at a port on or before the day of arrival in port.

      The interface is based on a WCF web service which fits into the Disney Cruise Lines technical infrastructure, and transfers data for a vessel on a daily update basis. The Purser will initiate printing of boarding passes upon or before arrival in a port, or when a mid-journey cabin transfer occurs (see below).
